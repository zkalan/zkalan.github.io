# 词性标注与隐马尔科夫模型
## 词性标注
### 定义和任务描述
什么叫词性？词性又称词类，是指词汇基本的语法属性。划分词类的依据包括词的形态、词的语法功能、词的语法意义。汉语的词类划分借用了英语的词类体系，但是汉语没有词性的变化。**词性标注**就是给语言的词标注上其所属的词类。
### 词性标注的问题——标注歧义
一个词具有两个以上的词性。
### 词性标注的重要性
是句法分析的预处理过程。机器翻译需要词性标注。文本—语言转换需要词性标注。
### 词性标注方法
- 规则方法
  - 词典提供候选词性
  - 人工整理标注规则
- 基于错误驱动的方法
  - 错误驱动学习规则
  - 利用规则重新标注词性
- 统计方法
  - 问题的形式化描述
  - 建立统计模型
    - HMM方法
    - 最大熵方法
    - 条件随机场
    - 结构化支持向量机
- 性能指标：标注准确率
  - 当前方法正确率可达97%
  - 正确率基准可达90%（给每个词标上它最常见的词性，所有未登录词标为名词）

![p11](image/hmm1.png)
$P(s,w)=\prod_iP(s_i|s_i-1)P(w_i|s_i)$
### 马尔可夫过程
- 一个系统有N个有限状态$S=\{s_1,s_2,...s_N\}$
- $Q=(q_1,q_2,...q_t)$是一个随机变量序列。随机变量的取值为状态集S中的某个状态
- $P(q_t=s_j|q_{t-1}=s_i,q_{t-2}=s_k,...,q_1=s_h)$
#### 有限视野假设
$P(q_{t+1}=s_k|q_1,q_2,...q_t)=P(q_{t+1}=s_k|q_{t-(n-1)},...,q_t)$
n-1阶马尔可夫链即为n元语言模型。
#### 时间独立性
$P(q_{t+1}=s_k|q_t=s_h)=P(q_{t+k+1}=s_k|q_{t+k}=s_h)$
**HMM是一阶马尔可夫模型的扩展**，隐藏的状态序列满足一阶马尔可夫模型，观察值与状态之间满足概率关系。相对于马尔可夫模型，增加一个假设：**输出独立性**
$P(O_1,...O_T|S_1,...S_T)=\prod_TP(O_t|S_t)$
该假设最大缺点在于没有考虑上下文，限制了特征的选择。
### HMM定义
- 定义：一个HMM模型$\lambda=(S,V,A,B,\Pi)$
- S是状态集，$S=(S_1,S_2，。。。S_N)$
- V是观察集，$V=(V_1,V_2，。。。V_M)$
- 状态序列$Q=q_1q_2...q_T$隐藏，观察序列$O=o_1o_2...o_T$可见
- A是状态转移概率分布$A=[a_{ij}]$，$a_{ij}=P(q_t=s_j|q_{t-1}=s_i)$满足假设一
- B是观察值生成概率分布$B=[b_j(v_k)]$，$b_j(v_k)=P(o_t=v_k|q_t=s_i)$满足假设二、三
- 初始状态概率分布$\Pi=[\Pi_i],\Pi_i=P(q_1=s_i)$
## HMM模型用于词性标注
- S：预先定义的词性标注集
- V：文本中的词汇
- A：词性之间的转移概率
- B：某个词性生成某个词的概率
- $\Pi$：初始概率
基于构建的HMM，利用某种算法，寻找一个最合适的词性标注序列，即为一个词串上的每个词标注上词性。
##相关问题
- 给定一个模型，如何计算某个特定的输出序列的概率？前向后向算法，前向意思就是从时刻1到时刻T，后向意思就是从时刻T到时刻1
- 给定一个模型和某个特定的输出序列，如何找到最可能产生这个输出的状态序列？维特比算法——动态规划算法——时间复杂度$O(N^2T)$
- 给出足够的观测数据，如何估计隐马尔可夫模型的参数？训练隐马尔可夫模型更实用的方式是仅仅通过大量观测到的信号$O_1,O_2,O_3,...$就能推算模型参数的$P(S_t|S_{t-1})$和$P(O_t|S_t)$的无监督训练方法，主要使用的是鲍姆-韦尔奇算法。

## 模型参数学习
- 给定状态集S和观察集V，学习模型参数A，B，$|Pi$
- 模型参数学习过程就是构建模型的过程
- 有指导学习——最大似然估计
![p28](image/ll.png)
- 无指导学习——Welch-Baum
![p29](image/wb.png)
基本思想：随机给出模型参数的初始化值，得到最初的模型$\lambda_0$，然后利用初始模型$\lambda_0$得到某一状态转移到另一状态的期望次数，然后利用期望次数对模型进行重新估计，由此得到模型$\lambda_1$，如此循环迭代，重新估计，直到模型参数收敛。

## 最大熵马尔可夫模型MEMM
- HMM：最大缺点在于没有考虑上下文，限制了特征的选择；计算联合概率；除了生成标签，还生成了单词，实际上只需要预测标签
- MEMM：直接计算条件概率；预测标记，不需要考虑输入的分布；容易引入各种特征，局部最优解，标记偏置；局部归一化
- CRF：解决标记偏置，无向图模型，不用概率模型解释；忍让是条件概率模型，不是对每个状态做归一化，而是对整个串或图做归一化，避免标记偏置
